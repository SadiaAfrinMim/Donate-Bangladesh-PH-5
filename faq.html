what is the document object model(Dom)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page so that programs (like JavaScript) can manipulate the content, structure, and styling of a webpage dynamically.

What is event delegration in the context of the DOM,and why is it useful?

Event delegation is a technique in JavaScript that allows you to handle events at a higher level in the DOM than the target element, making it possible to manage events more efficiently. Instead of adding individual event listeners to many child elements, you can add a single event listener to a parent element, and let the event "bubble up" from the child to the parent.
